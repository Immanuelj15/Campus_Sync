# Campus Sync 🎓
### *Connecting Campus Communities, One Hand at a Time.*

[![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)](https://flutter.dev)
[![Firebase](https://img.shields.io/badge/firebase-%23039BE5.svg?style=for-the-badge&logo=firebase)](https://firebase.google.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

**Campus Sync** is a powerful peer-to-peer support platform built with Flutter and Firebase, designed specifically for university students. It bridges the gap between students who need help and those who can provide it, fostering a culture of collaboration, mutual assistance, and community building within campus boundaries.

---

## 🚀 Key Features

- **🧠 AI-Driven Categorization**: Automatically detects request types (Tech, Study, Travel, Food, Stay) and urgency levels from simple text descriptions, ensuring the right students see your request instantly.
- **💬 Contextual Private Chat**: Integrated one-on-one messaging system tied specifically to each request. No need to share personal numbers; just sync and chat.
- **🏆 Gamified Trust System**: A dynamic badge system (e.g., *Helpful Human*, *Campus Connector*) that rewards community contributors and builds a trusted peer network.
- **🛡️ Secure Peer Support**: Optional anonymous posting for sensitive requests, plus built-in moderation and reporting tools to keep the campus safe.
- **📈 Resolve & Rate**: A closed-loop system where helpers are recognized with "Thank You" notes and ratings, ensuring quality and accountability.
- **🔔 Real-time Sync**: Instant push notifications for help offers, messages, and status updates.

---

## 💡 Why Campus Sync?

### How it helps students:
- **Academic Support**: Get quick access to study materials, notes, or peer tutoring before exams.
- **Logistical Ease**: Find a lift, borrow a charger, or get help with moving dorms.
- **Financial & Resource Sharing**: Borrow or exchange books and resources within the community.
- **Meaningful Connections**: Meet fellow students through helpful interactions, turning a large campus into a close-knit community.

### What makes it unique:
Unlike generic social networks, Campus Sync is **task-oriented and trust-based**. It's not about likes; it's about **impact**. The intelligent categorization system reduces noise, making sure you only see requests you can actually help with or those that matter most to you.

## 🛠️ Tech Stack

- **Frontend**: Flutter (Dart) for high-performance cross-platform UI.
- **Backend**: Firebase (Firestore, Auth, Storage, Cloud Functions).
- **Architecture**: Modular Service-based architecture with Stream-driven UI updates.
- **Notifications**: Firebase Cloud Messaging (FCM).

## 🛠️ Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Firebase Account](https://firebase.google.com/)

### Installation
1. **Clone the repo**
   ```bash
   git clone https://github.com/Immanuelj15/Campus_Sync.git
   ```
2. **Install dependencies**
   ```bash
   flutter pub get
   ```
3. **Setup Firebase**
   - Create a Firebase project at [Firebase Console](https://console.firebase.google.com/).
   - Add your Android/iOS app.
   - Run `flutterfire configure` to generate `firebase_options.dart`.
4. **Run the app**
   ```bash
   flutter run
   ```
